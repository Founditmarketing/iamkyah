# IAmKyah.com Website

This is the portfolio and professional website for Kyah Spriggs—advocate, speaker, and suicide survivor. The platform highlights her mission to save lives through the "Six Feet Above" nonprofit and offers a booking system for speaking engagements.

## Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS
- Animation: Framer Motion
- Icons: Lucide React

## Run Locally

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Build for production:
   `npm run build`
